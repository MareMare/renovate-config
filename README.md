# renovate-config
_⚠️ This repository is intended for me._

Shareable Renovate Config Presets.

Renovate 設定の共有プリセットです。

## 使い方
共有プリセット `default.json` を使用する他のリポジトリでは `renovate.json` を以下のようにします。
```json
{
  "extends": [
    "github>MareMare/renovate-config"
  ]
}
```
